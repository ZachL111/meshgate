# Meshgate Walkthrough

The fixture is intentionally compact, so the review starts with the cases that pull farthest apart.

| Case | Focus | Score | Lane |
| --- | --- | ---: | --- |
| baseline | rollout width | 193 | ship |
| stress | quota pressure | 191 | ship |
| edge | route drift | 224 | ship |
| recovery | secret scope | 175 | ship |
| stale | rollout width | 198 | ship |

Start with `edge` and `recovery`. They create the widest contrast in this repository's fixture set, which makes them better review anchors than the middle cases.

The useful comparison is `route drift` against `secret scope`, not the raw score alone.
